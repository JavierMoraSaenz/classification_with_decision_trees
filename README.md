
# Clasificación de Medicamentos con Árboles de Decisión

Proyecto de Machine Learning supervisado utilizando Árboles de Decisión para predecir el tipo de medicamento adecuado según características clínicas de los pacientes.

## 📌 Objetivo

Construir y evaluar modelos de clasificación capaces de identificar el medicamento más apropiado utilizando variables demográficas y médicas.

## 🛠 Tecnologías utilizadas

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Graphviz
- Jupyter Notebook

## 📊 Temas desarrollados

### 🔹 Preprocesamiento de datos
- Codificación de variables categóricas
- Preparación de variables predictoras

### 🔹 Árboles de Decisión
- Entrenamiento del modelo
- Predicción de clases
- Comparación de criterios Gini y Entropy

### 🔹 Evaluación del modelo
- Matriz de confusión
- Accuracy
- Precision
- Recall
- F1-Score

### 🔹 Interpretabilidad
- Generación de reglas de decisión
- Visualización gráfica del árbol

## 📈 Resultados

Se entrenaron modelos de clasificación para identificar el medicamento adecuado a partir de variables como edad, sexo, presión arterial, colesterol y relación sodio-potasio.

## 🚀 Aprendizajes

- Implementación de árboles de decisión con Scikit-learn
- Comparación entre índices Gini y Entropy
- Evaluación de modelos de clasificación
- Interpretación de reglas de decisión
- Visualización de estructuras de decisión
